# OutSource

### It's hard enough to get outside and touch grass. OutSource uses your preferences, your circle, and Google Gemini to recommend you activities when the moment is right.

---

To run flutter app:
1. download and install project dependencies
2. run docker
3. navigate to outsource/backend
4. on command line, enter `docker compose up`
5. navigate to outsource/frontend
2. on command line, enter `flutter run`
